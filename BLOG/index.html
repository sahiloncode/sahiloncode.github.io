📖 History & Experiences

🚀 How It All Began

My journey into the world of development began with a curiosity for how websites work behind the scenes. What started as simple HTML and CSS experiments soon grew into a strong passion for creating clean, functional, and user-friendly web applications. Over time, I found myself exploring JavaScript, responsive design, and frontend frameworks.

🧠 Learning Through Projects

I firmly believe that real learning comes from building. From my first static site to dynamic JavaScript projects and functional clones of real-world apps like Amazon and Myntra, each project taught me something new — from logic structuring to UI/UX decisions. These hands-on experiences refined my ability to think critically and solve problems efficiently.

🛠️ Challenges Faced

Like every developer, I faced countless bugs, design roadblocks, and even burnout. But each obstacle pushed me to dig deeper, improve my problem-solving approach, and stay consistent. Whether it was fixing cross-browser issues or managing responsive layouts, these challenges became milestones in my journey.

📌 Present Focus

Currently, my focus is on mastering modern frontend technologies like Tailwind CSS, JavaScript, and building polished, scalable UI experiences. I’m also exploring the basics of backend integration and Firebase to become more versatile as a developer. Sharing what I learn has become just as rewarding as building itself.

🌟 What’s Next

My long-term goal is to contribute to meaningful projects, collaborate with passionate developers, and never stop growing. I'm actively improving my portfolio, writing technical blogs, and staying up-to-date with emerging technologies to stay ahead in this dynamic field.

📌 VS Code Tips & Extensions

🎨 Prettier

Prettier is an opinionated code formatter that enforces a consistent style by parsing your code and reprinting it. It removes all original styling and ensures uniform code layout across the entire team.

🔍 GitLens

GitLens supercharges Git in VS Code. It helps you visualize code authorship via Git blame annotations, explore repository history, and understand who, why, and when a line of code was changed.

⚡ Live Server

Launch a local development server with live reload for static and dynamic pages. Ideal for frontend developers to preview HTML/CSS/JS changes in real-time.

🤖 Tabnine

Tabnine uses machine learning to autocomplete code as you type. It supports multiple languages and frameworks, helping you write code faster with fewer errors.

🧹 ESLint

ESLint statically analyzes your code to quickly find problems and enforces coding style rules. It's a must-have for JavaScript and TypeScript projects to maintain clean code.

📡 REST Client

REST Client allows you to send HTTP requests and view responses directly inside VS Code. No need to switch to Postman or another API tool — great for backend or API developers.

📁 Path Intellisense

This extension autocompletes filenames and paths in your import statements, saving you time and preventing errors in file navigation or linking resources.

🟩 Bracket Pair Colorizer 2

It highlights matching brackets with different colors, making it easier to spot bugs and understand nested code blocks — especially in JavaScript, Python, or C-like languages.

📸 CodeSnap

Take beautiful screenshots of your code directly from VS Code. It’s perfect for sharing on social media, documentation, or tech blogs.

🧠 IntelliCode

Microsoft IntelliCode provides AI-assisted code suggestions based on best practices learned from thousands of open-source repositories. Improves speed and accuracy while coding.
